Amazon S3

Simple Storage Service

O Amazon S3 é o serviço de armazenamento de objetos da AWS. Ele permite armazenar qualquer quantidade de arquivos com alta disponibilidade, segurança e escalabilidade.

📦 O que é um Bucket?

Buckets são contêineres utilizados para armazenar objetos dentro do Amazon S3.

📁 imagens

📁 documentos

📁 videos
