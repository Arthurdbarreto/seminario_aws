O que é S3?

Visão geral

O Amazon S3 é o serviço de armazenamento de objetos da AWS. Ele é utilizado para armazenar arquivos como imagens, vídeos, documentos e backups.

Um bucket funciona como um contêiner para objetos.

Dentro dele, cada arquivo armazenado no S3 é tratado como um objeto, com nome, dados e metadados próprios.
