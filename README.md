# ☁️ AWS Learning Portal

Projeto desenvolvido para o seminário da disciplina **Computação em Nuvem e Web Services**, com o objetivo de apresentar conceitos fundamentais da computação em nuvem e demonstrar alguns dos principais serviços da **Amazon Web Services (AWS)** por meio de uma experiência interativa em formato de portal web.

---

## 📖 Sobre o Projeto

O **AWS Learning Portal** foi criado para transformar uma apresentação tradicional em uma experiência visual e interativa.

Em vez de utilizar apenas slides, o grupo desenvolveu um portal web inspirado na identidade visual da AWS, permitindo a navegação entre diferentes serviços e conceitos relacionados à computação em nuvem.

O projeto busca apresentar conteúdos teóricos de forma clara, moderna e acessível, simulando a experiência de uma plataforma educacional corporativa.

---

## 🎯 Objetivos

* Compreender os conceitos fundamentais de Computação em Nuvem.
* Apresentar os principais modelos de implantação de nuvem.
* Demonstrar o funcionamento dos serviços AWS.
* Criar uma experiência visual moderna para apoio ao seminário.
* Aplicar conceitos de desenvolvimento web utilizando HTML, CSS e JavaScript.

## 📚 Conteúdo Apresentado

### ☁️ Cloud Computing

Apresenta os conceitos fundamentais da computação em nuvem:

* O que é Cloud Computing
* Evolução da infraestrutura de TI
* Nuvem Pública
* Nuvem Privada
* Nuvem Híbrida
* Benefícios da Computação em Nuvem
* Arquitetura AWS

---

### 💻 Amazon EC2

Página dedicada ao serviço de computação da AWS.

Conteúdos abordados:

* O que é Amazon EC2
* Instâncias Virtuais
* Elasticidade
* Auto Scaling
* Dashboard Simulado
* Casos de Uso
* Benefícios da Computação Escalável

---

### 📦 Amazon S3

Página dedicada ao serviço de armazenamento da AWS.

Conteúdos abordados:

* Conceitos de Buckets e Objetos
* Estrutura de Armazenamento
* Classes de Armazenamento
* Durabilidade e Disponibilidade
* Casos de Uso Corporativos
* Backups e Hospedagem de Arquivos

---

### 🔐 AWS IAM

Página dedicada ao serviço de gerenciamento de identidades e acessos.

Conteúdos abordados:

* Usuários
* Grupos
* Policies
* Roles
* MFA (Multi-Factor Authentication)
* Princípio do Menor Privilégio
* Controle de Permissões
